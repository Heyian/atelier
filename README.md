# Atelier

*[English follows](#atelier--english)*

Un ensemble de compétences (« skills ») Claude pour les dirigeants qui veulent
un Claude qui travaille comme eux.

Atelier donne aux leaders non techniques un point de départ solide —
intégration guidée, compétences prêtes à l'emploi pour le marketing, les
opérations de vente et les réunions, plus un assistant de création de
compétences — pour qu'ils finissent avec une boîte à outils faite pour eux,
par eux.

Conçu pour Claude Cowork et l'application Claude Desktop. Aucun terminal requis.

## Compétences

| Compétence | Rôle |
|------------|------|
| `atelier` | Le cœur : entrevue d'intégration, profil d'entreprise, relais de session, et l'art de travailler avec Claude (délégation, gestion du contexte) |
| `atelier-mentor` | Le guide : oriente vers la bonne compétence et conseille sur les meilleures pratiques d'utilisation de l'IA |
| `atelier-marketing` | Création de contenu, planification de campagnes, capture de la voix de marque |
| `atelier-ventes` | Revues de pipeline, suivis, propositions, hygiène CRM |
| `atelier-reunions` | Cycle complet des réunions : préparation, procès-verbaux (PV), journaux de décisions, communications |
| `atelier-forge` | Le créateur de compétences : une entrevue en langage clair qui génère vos propres compétences sur mesure |

Chaque compétence est offerte en français et en anglais.

# Atelier — English

A Claude skill pack for executives who want Claude working the way they work.

Atelier gives non-technical leaders a strong starting point — guided
onboarding, ready-made skills for marketing, sales operations, and meetings,
plus a guided skill-builder — so they end up with a toolkit made for them, by
them.

Built for Claude Cowork and the Claude Desktop app. No terminal required.

## Skills

| Skill | Purpose |
|-------|---------|
| `atelier` | The hub: onboarding interview, company profile, session handoff, and the Claude-craft curriculum (delegation, context management) |
| `atelier-mentor` | The guide: routes you to the right skill and advises on AI best practices and workflow optimization |
| `atelier-marketing` | Content creation, campaign planning, brand-voice capture |
| `atelier-sales` | Pipeline reviews, follow-ups, proposals, CRM hygiene |
| `atelier-meetings` | Full meeting lifecycle: prep, minutes, decision logs, communications |
| `atelier-forge` | The skill-builder: a plain-language interview that generates their own custom skills |

Every skill ships in both French and English.

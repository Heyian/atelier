# Atelier

*[English follows](#atelier--english)*

Un ensemble de compétences (« skills ») Claude pour les dirigeants qui veulent
un Claude qui travaille comme eux.

Atelier donne aux leaders non techniques un point de départ solide —
intégration guidée, compétences prêtes à l'emploi pour le marketing, les
opérations de vente et les réunions, plus un assistant de création de
compétences — pour qu'ils finissent avec une boîte à outils faite pour eux,
par eux.

Conçu pour Claude Cowork et l'application Claude Desktop. Aucun terminal requis.

## Compétences

| Compétence (FR) | Skill (EN) | Rôle |
|------------------|------------|------|
| `atelier` | `atelier` | Le cœur : entrevue d'intégration, profil d'entreprise, relais de session, et l'art de travailler avec Claude (délégation, gestion du contexte) |
| `atelier-mentor` | `atelier-mentor` | Le guide : t'oriente vers la bonne compétence et te conseille sur les meilleures pratiques d'utilisation de l'IA |
| `atelier-marketing` | `atelier-marketing` | Création de contenu, planification de campagnes, capture de la voix de marque |
| `atelier-ventes` | `atelier-sales` | Revues de pipeline, suivis, propositions, hygiène CRM |
| `atelier-reunions` | `atelier-meetings` | Cycle complet des réunions : préparation, procès-verbaux (PV), journaux de décisions, communications |
| `atelier-forge` | `atelier-forge` | Le créateur de compétences : une entrevue en langage clair qui génère tes propres compétences sur mesure |
| `atelier-boussole` | `atelier-compass` | Le processus de réflexion : nomme la destination, cartographie les décisions, une décision par conversation |

Chaque compétence est offerte en français et en anglais. Guide d'installation :
[`docs/INSTALL.fr.md`](docs/INSTALL.fr.md) · [`docs/INSTALL.en.md`](docs/INSTALL.en.md).

Télécharge la [dernière version](https://github.com/Heyian/atelier/releases/latest).

# Atelier — English

A Claude skill pack for executives who want Claude working the way they work.

Atelier gives non-technical leaders a strong starting point — guided
onboarding, ready-made skills for marketing, sales operations, and meetings,
plus a guided skill-builder — so they end up with a toolkit made for them, by
them.

Built for Claude Cowork and the Claude Desktop app. No terminal required.

## Skills

| Skill (EN) | Compétence (FR) | Purpose |
|------------|------------------|---------|
| `atelier` | `atelier` | The hub: onboarding interview, company profile, session handoff, and the Claude-craft curriculum (delegation, context management) |
| `atelier-mentor` | `atelier-mentor` | The guide: routes you to the right skill and advises on AI best practices and workflow optimization |
| `atelier-marketing` | `atelier-marketing` | Content creation, campaign planning, brand-voice capture |
| `atelier-sales` | `atelier-ventes` | Pipeline reviews, follow-ups, proposals, CRM hygiene |
| `atelier-meetings` | `atelier-reunions` | Full meeting lifecycle: prep, minutes, decision logs, communications |
| `atelier-forge` | `atelier-forge` | The skill-builder: a plain-language interview that generates their own custom skills |
| `atelier-compass` | `atelier-boussole` | The thinking process: name the destination, map the decisions, one decision per conversation |

Every skill ships in both French and English. Install guides:
[`docs/INSTALL.en.md`](docs/INSTALL.en.md) · [`docs/INSTALL.fr.md`](docs/INSTALL.fr.md).

Download the [latest release](https://github.com/Heyian/atelier/releases/latest).

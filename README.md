# Atelier

A skill pack for executives who want Claude working the way they work.

Atelier gives non-technical leaders a strong starting point — onboarding, ready-made
skills for marketing, sales operations, and meetings, plus a guided skill-builder so
they end up with a toolkit made for them, by them.

Built for Claude Cowork and the Claude Desktop app. No terminal required.

## Skills

| Skill | Purpose |
|-------|---------|
| `atelier` | The hub: first-run onboarding, company profile, and the Claude-craft curriculum (delegation, session handoffs, context management) |
| `atelier-marketing` | Content creation, campaign planning, brand-voice capture |
| `atelier-ventes` | Pipeline reviews, follow-ups, proposals, CRM hygiene |
| `atelier-reunions` | Meeting prep, note processing, decision logs, communications |
| `atelier-forge` | The skill-builder: a plain-language interview that generates their own custom skills |

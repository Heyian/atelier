# What to delegate, and what stays yours

When the question sounds like: "what can I actually hand off to Claude?",
"how far can I go without watching it?"

**Draw the human/AI line, function by function.** The executive keeps
judgment, the client relationship, final sign-off. Claude does research,
drafting, analysis. Write the line down somewhere — even three lines — rather
than re-deciding it for every task.

**Every outward-facing output passes through a review gate the executive
owns.** No new process needed: the review task in whatever tool is already in
use (ClickUp, Teams, an email draft) IS the gate, and it leaves an automatic
trail.

**Permissions follow the phase, not the ambition.** Let Claude read a system
before letting it write to it; widen deliberately, never by drift — especially
for anything touching shared documents or connected tools (SharePoint, a
CRM).

**Typical next practice:** if nothing is written down about who decides what,
offer to write that line now, for the function already under discussion.

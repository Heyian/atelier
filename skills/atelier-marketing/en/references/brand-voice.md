# Capturing the brand voice, then scoring against it

A voice guide only earns its keep if it's checkable — not an adjective you
wave around, but a list you can tick off.

## Step 1 — Gather real samples

Five to ten pieces that actually went out: sent newsletters, posts, website
pages, a sales email that landed well. Not copy you wish you'd written —
copy you actually sent, warts included.

Few or no samples on hand? Start from the Company Profile's Tone of voice
and Vocabulary sections. That's a fair starting point — the three adjectives
and banned words are already there — but it isn't a full checklist yet: say
so to the executive, and offer to sharpen it once real samples exist.

## Step 2 — Pull four axes out of what you're reading

- **Words** — the ones that keep showing up, the ones you never see.
- **Rhythm** — short, clipped sentences, or ones that run on and stack up?
- **Stance** — direct or inviting, does it give orders or ask questions,
  does it joke or stay serious?
- **Off-limits** — what this brand never does: a pun, technical jargon, an
  exclamation point, a word in all caps.

## Step 3 — Turn each axis into a checkable test

"Warm" isn't checkable. "Uses the reader's name, asks a direct question in
the first two sentences" is. If a criterion can't get a plain yes or no on
rereading, it isn't finished yet — rephrase it.

Sample checklist, to be shaped by what the samples actually showed — not
copied as-is:

| Criterion | Yes / No |
|---|---|
| Addresses the reader directly | |
| A direct question in the first two sentences | |
| No exclamation points | |
| None of the profile's banned words | |
| Average sentence length under ten words | |
| Closes on exactly one requested action | |

## Step 4 — Write it, then get it confirmed

The guide lives at `{root}/docs/marketing/guide-de-voix.md`. It's living
state, not a one-off deliverable: reread at the start of each new draft,
then updated rather than rebuilt — and a change is proposed before it's
written, same as everywhere else in Atelier. Show it to the executive and
ask: "does this sound like you?" — a guide they don't recognize is dead
weight, however well built. One correction from them is worth more than ten
criteria guessed alone.

## Score a draft — don't judge it by feel

This is what makes the guide worth having: never reread a draft asking
whether "it sounds right." Read it line by line against the checklist
instead.

1. Take each checklist item, one at a time.
2. Mark it pass or fail on this specific draft — never an overall
   impression.
3. An item fails: rewrite that line, and show the before and after if it
   helps the executive see the difference.
4. Never deliver copy unscored when a voice guide exists — scoring is part
   of the delivery, not an optional step that gets dropped under time
   pressure.
